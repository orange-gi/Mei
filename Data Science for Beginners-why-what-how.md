# Data Science for Beginners 项目介绍

> Microsoft Azure Cloud Advocates 提供的 10 周、20 课时数据科学入门课程

---

## 一、为什么（Why）

### 1.1 项目背景与动机

在当今数据驱动的世界中，数据科学已成为各行各业的关键技能。然而，对于初学者而言，数据科学领域往往显得庞大且令人生畏，难以找到系统性的入门路径。Microsoft 观察到大量希望学习数据科学的学习者面临着以下挑战：

- **知识体系碎片化**：网络上关于数据科学的资源众多，但缺乏系统性的学习路径，学习者往往东学一点、西学一点，难以形成完整的知识体系。
- **理论与实践脱节**：许多教程偏重理论讲解，缺乏实际项目经验，导致学习者无法将知识应用于真实场景。
- **语言障碍**：优质的数据科学课程大多以英文为主，对非英语母语的学习者不够友好。
- **学习动力难以维持**：传统课程缺乏互动性和成就感，学习者容易中途放弃。

基于这些观察，Microsoft 决定打造一套专门为零基础学习者设计的课程体系，帮助更多人踏入数据科学的大门。

### 1.2 解决的核心问题

本项目致力于解决以下核心问题：

| 问题 | 解决方案 |
|------|----------|
| 学习路径不清晰 | 提供 10 周、20 课时的系统化课程，按难度循序渐进 |
| 缺乏实践机会 | 采用项目导向教学法，每节课都有配套项目 |
| 语言障碍 | 提供 50+ 种语言支持，包括中文（简体/繁体） |
| 学习效果难以评估 | 提供课前热身测验和课后测验，共 40 个测验确保知识保持 |
| 难以持续学习 | 灵活的课程设计，可整体学习或选择性学习 |

### 1.3 目标用户群体

本课程面向以下人群：

- **完全零基础者**：对数据科学一无所知，希望从最基础的概念开始学习
- **转行者**：希望从其他领域转向数据科学职业道路
- **学生**：希望在课程学习之外拓展数据科学知识
- **教育工作者**：需要为学生提供数据科学入门课程资源的教师
- **业务人员**：工作中需要理解数据科学概念和数据驱动决策的业务人员

### 1.4 为什么现有方案不够好

虽然市面上存在大量的数据科学学习资源，但本课程具有以下独特优势：

1. **权威性背书**：由 Microsoft Azure Cloud Advocates 团队开发，质量有保障
2. **系统性设计**：从概念入门到云端部署，覆盖完整的数据科学生命周期
3. **项目导向学习**：通过动手构建项目来学习，这是已被证明最有效的学习方式
4. **多语言支持**：支持 50+ 种语言，打破语言障碍
5. **社区支持**：提供 Discord 社区和 GitHub Discussions 讨论区，学习过程不孤单
6. **免费开源**：完全免费，采用 MIT 许可证，可自由使用和分发

## 二、是什么（What）

### 2.1 项目概述

**Data Science for Beginners** 是由 Microsoft Azure Cloud Advocates 提供的开源数据科学入门课程。该课程采用项目导向教学法（Project-based Pedagogy），让学习者在实践中学习数据科学技能。

| 项目指标 | 数值 |
|----------|------|
| 课程时长 | 10 周 |
| 课时数量 | 20 课时 |
| GitHub Stars | 32,300+ |
| GitHub Forks | 6,800+ |
| 贡献者 | 118+ |
| 支持语言 | 50+ 种 |

### 2.2 核心功能与特性

#### 🎓 教学方法

- **项目导向学习 (Project-based Learning)**：通过构建真实项目来学习新技能，这是已被证明能让新技能「固化」的有效方法
- **频繁测验机制**：每节课包含课前热身测验和课后测验，确保知识保持
- **灵活学习模式**：可选择整体学习或选择性学习特定主题
- **循序渐进的项目**：项目从简单到复杂，逐步深入

#### 📚 课程内容结构

每节课包含以下组件：

1. **可选的手绘笔记 (Sketchnote)**：直观的视觉总结，帮助理解核心概念
2. **可选的补充视频**：提供视频学习选项，适应不同学习偏好
3. **课前热身测验**：设定学习预期，激发学习兴趣
4. **书面教程**：详细的学习材料
5. **项目构建指南**：逐步指导完成项目
6. **知识检测**：检查理解程度
7. **挑战任务**：拓展练习
8. **补充阅读材料**：深度学习资源
9. **课后作业**：巩固所学知识
10. **课后测验**：40 个测验，每个包含 3 道题目

### 2.3 主要特性

- **完整的数据科学知识体系**：从基础概念到云端部署全覆盖
- **真实世界案例**：第 20 课专门介绍真实世界中的数据科学项目
- **云端学习支持**：可使用 GitHub Codespaces 或 VSCode Remote-Containers 进行学习
- **离线访问支持**：可通过 Docsify 在本地运行文档
- **教师指南**：为教育工作者提供教学指导

### 2.4 与竞品的差异化

| 维度 | Data Science for Beginners | 典型竞品 |
|------|---------------------------|----------|
| 权威性 | Microsoft 官方团队开发 | 个人或小型团队开发 |
| 语言支持 | 50+ 种语言 | 通常仅支持英语 |
| 教学方法 | 项目导向 + 频繁测验 | 纯理论或单一项目 |
| 覆盖范围 | 完整生命周期 | 往往只覆盖部分内容 |
| 社区支持 | Discord + GitHub Discussions | 通常仅有 GitHub Issues |
| 更新维护 | Microsoft 持续维护 | 可能已停止维护 |

### 2.5 技术栈概述

虽然本课程主要面向初学者，但涉及的技术栈包括：

- **编程语言**：Python（主要教学语言）、SQL
- **数据处理库**：Pandas、NumPy
- **数据可视化**：Matplotlib、Seaborn
- **数据库**：关系型数据库（SQL）、非关系型数据库（NoSQL）
- **云平台**：Microsoft Azure Machine Learning Studio
- **低代码工具**：Azure ML Designer
- **开发环境**：VS Code、GitHub Codespaces、Docker

## 三、怎么做（How）

### 3.1 快速开始指南

#### 方式一：使用 GitHub Codespaces（推荐）

```bash
1. 访问课程仓库：https://github.com/microsoft/Data-Science-For-Beginners
2. 点击 "Code" 按钮，选择 "Open with Codespaces"
3. 点击 "+ New codespace" 创建新环境
4. 环境启动后即可开始学习
```

#### 方式二：本地克隆（不含翻译）

由于翻译文件较大，建议使用稀疏克隆：

```bash
# 克隆仓库（不包含翻译文件）
git clone --filter=blob:none --sparse https://github.com/microsoft/Data-Science-For-Beginners.git
cd Data-Science-For-Beginners
git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
```

#### 方式三：使用 VSCode Remote-Containers

```bash
1. 确保已安装 Docker 和 VSCode Remote-Containers 扩展
2. 克隆仓库到本地
3. 在 VSCode 中按 F1，选择 "Remote-Containers: Open Folder in Container..."
4. 选择克隆的文件夹，等待容器启动
```

#### 方式四：离线访问

```bash
# 安装 docsify
npm install -g docsify

# 在项目根目录运行
docsify serve

# 访问 http://localhost:3000
```

- 20 | 真实世界示例 - 真实世界中的数据科学驱动项目

- 什么是数据科学？

**学习目标**：
